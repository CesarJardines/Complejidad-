Solo basta con tener python3 para que se ejecute el programa.

Ejecutar: python3 3SAT.py 

El programa asignara a 10 variables valores entre el 0 y el 1
Despues acomodará cualquiera de esas 10 variables en 5 clausulas (las variables se pueden repetir)
La fase verificadora se constrara diciendote si ES o NO ES satisfacible la propuesta de solución de la 
fase adivinadora. 


Ejecutar: python ArbolGenerador.py

El programa construirá una gráfica con 5 vértices y a cada uno de esos vértices
le asignará aristas de manera aleatorea, despues el programa tomará una subgráfica
con los vértices y aristas que haya construido y verificará si el ejemplar es 
satisfacible o no.
